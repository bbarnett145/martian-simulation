# martian-simulation
System involving Martians (aliens) that displays features such as inheritance, ArrayLists, polymorphism, etc.
UML Diagram is attached.

GreenMartians have an ID and Volume. They are also Teleporters.
RedMartians have an ID, Volume, and Tenacity.
Martians have methods in the a MartianManger where a few methods demonstrating operations on the Martians exist.
